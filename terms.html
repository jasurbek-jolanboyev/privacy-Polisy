8. How the App Works

File Scanning

Files are analyzed locally using YARA rules (200+ malware patterns). Only the file hash (MD5/SHA256) is sent to VirusTotal for cloud verification – the file itself is never uploaded. Threat levels: CRITICAL / HIGH / MEDIUM / LOW / CLEAN.

Wi-Fi Security (Pro)

The app scans nearby networks and detects Evil Twin access points (rogue Wi-Fi hotspots that mimic legitimate networks). A push notification is sent when a threat is detected.

Background Monitoring (Pro)

The app monitors file system changes and Wi-Fi events in the background. On Android 12+, battery optimization must be disabled for Himoya to ensure reliable background operation.

AI Assistant

The built-in AI assistant answers security questions, explains scan results, and provides device protection recommendations. Available in Uzbek, Russian, and English.
