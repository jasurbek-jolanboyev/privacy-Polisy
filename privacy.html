Privacy Policy

App: Himoya – DLP Security Suite
Last updated: June 18, 2025
Platforms: Android (Google Play) · iOS (App Store)

1. Introduction

This Privacy Policy explains how Himoya – DLP Security Suite ("we", "our", or "the app") collects, uses, and protects information when you use our mobile application. By downloading or using the app, you agree to this policy.

2. Information We Collect

Account information

  • Email address – for login and notifications
  • Password – stored in encrypted form only (Firebase Auth)
  • Display name – optional, for profile purposes

Device information

  • Device identifier – for license management
  • OS version – for compatibility checks
  • App version – for troubleshooting
  • FCM token – for push notifications

Scan data

  • File metadata (name, size, type) – for threat analysis only
  • File hash (MD5/SHA256) – sent to VirusTotal for cloud verification (Pro tier)
  • Wi-Fi network info (SSID, signal strength) – for security analysis (Pro tier)
  • Scan history – stored for 7 days (Free), 30 days (Pro), 90 days (Enterprise)

What we never collect

  • File contents (documents, images, videos)
  • Messages or SMS
  • Contacts
  • Location (GPS)
  • Financial or banking information

3. How We Use Information

Collected data is used only for:

  • Detecting malicious files using YARA rules and VirusTotal
  • Sending security alerts via Firebase Cloud Messaging
  • Displaying personal scan statistics
  • Improving app performance
  • Managing Pro and Enterprise licenses

We do not sell or rent user data to third parties for advertising.

4. Data Storage and Security

  • All data is stored on Google Cloud (Firebase) infrastructure with encryption at rest
  • All network connections use TLS/HTTPS encryption
  • Pro activation is handled exclusively by server-side Cloud Functions – client-side modification is not possible
  • Only the file hash is sent to VirusTotal – the actual file is never uploaded

5. Third-Party Services

  • Firebase (Google) – authentication, database, cloud functions, push notifications. Google Privacy Policy
  • VirusTotal – file hash verification. VirusTotal Privacy Policy
  • Google Mobile Ads – ads shown on the Free tier (uses advertising ID).

6. Android Permissions

  • READ_EXTERNAL_STORAGE – required for file scanning
  • INTERNET – required for Firebase and VirusTotal
  • ACCESS_WIFI_STATE – Wi-Fi security analysis (Pro)
  • CHANGE_WIFI_STATE – Evil Twin detection (Pro)
  • RECEIVE_BOOT_COMPLETED – background monitoring (Pro)
  • POST_NOTIFICATIONS – security alerts (optional)

7. Children's Privacy

The app is not directed at children under the age of 13. We do not knowingly collect personal information from children under 13. If you believe a child has provided us with personal data, please contact us and we will delete it immediately. This complies with COPPA and Google Play / App Store children's privacy requirements.

8. Data Retention

  • Account data – retained until the account is deleted
  • Scan history – 7 days (Free), 30 days (Pro), 90 days (Enterprise)
  • VirusTotal cache – 7 days
  • FCM token – until device is removed from account

9. User Rights

You have the right to:

  • Access your personal data
  • Edit your profile information within the app
  • Delete your account and all associated data
  • Opt out of notifications
  • Request a data export

To exercise these rights, contact us at privacy@himoya.uz. Requests are processed within 30 days.

10. International Data Transfer

Your data is stored on Firebase servers located primarily in the United States and Europe. Transfers are governed by Google's Standard Contractual Clauses (SCC) and applicable data protection laws.

11. Changes to This Policy

We may update this policy from time to time. When we do, we will notify users via in-app notification and email. Continued use of the app after changes constitutes acceptance of the updated policy.
